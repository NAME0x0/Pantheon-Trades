# Oracle Hall

The Oracle Hall is the web UI section where users can inspect all signal and thesis data — the "hall of prophecies."

## Pages

### `/signals` — Signal Band View
Live feed of all signals generated by Apollo, organized by band:
- Color-coded band indicators (S=gold, A=green, B=yellow, C=orange, D=grey)
- Edge score and direction (↑ YES, ↓ NO)
- Liquidity, catalyst, and sentiment sparkline indicators
- Time since last update
- Status: queued for deliberation / deliberating / expired / in paper mode

WebSocket: `ws://api/ws/signals` — live updates as signals arrive.

### `/theses` — Thesis List
All council deliberations:
- Thesis status badge (pending / approved / rejected / executed / expired)
- Market question (truncated)
- Council probability vs. market probability → edge
- Confidence score
- Agent vote summary (approve/reject/abstain counts)
- Timestamp

Filterable: by status, band, market category, date range.

### `/theses/{id}` — Thesis Detail
Full thesis view:
- Market question and metadata
- Council vote table (all agents, their votes, confidence, probability estimates)
- Edge chart: council_probability vs. market_probability over time
- Exit conditions
- Areopagus decision + notes
- Link to trace viewer

### `/traces` — Trace List
All deliberation traces:
- Search by market question, agent flag, date
- Quick scan: verdict + key flags

### `/traces/{hash}` — Trace Viewer
Full debate viewer:
- Timeline of all trace events
- Per-agent, per-round expandable view
- Veto and flag events highlighted
- Agent response latency chart
- Voting round visualization

### `/trades` — Trade List
All executed positions:
- Open vs. closed
- Real-time unrealized PnL for open positions
- Final PnL for closed positions
- Link to parent thesis

### `/leaderboard` — Agent Leaderboard
Council agent performance:
- Brier score (30-day rolling)
- Sharpe ratio (30-day)
- Win rate
- Total deliberations
- Restraint Quality section
- No-Trade Alpha section

### `/agents/{id}` — Agent Detail
Per-agent view:
- Full performance history
- Vote distribution (approve/reject/abstain over time)
- Calibration chart: stated confidence vs. actual accuracy
- Passport link (Arc Testnet explorer)

### `/settings` — System Settings
- Data source health (staleness dashboard)
- System state indicator
- Human review queue (operator+ only)
- Goals board

## Real-Time Updates

The Oracle Hall uses WebSocket connections for live data:
- Signal feed streams as new signals are scored
- Active deliberations show trace events in real-time
- Open position PnL updates every 30s
- Leaderboard refreshes after each market resolution
